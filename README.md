# TimeDensity — новая физическая модель времени как субстанции

**TimeDensity** — это научная теория, рассматривающая время как *физическую среду* с собственной плотностью, потоком и внутренней динамикой.  
В отличие от классических моделей, где время — параметр или координата, TimeDensity трактует его как **вещество**, участвующее в формировании гравитации, структуры пространства и эволюции Вселенной.

---

## Основные идеи

- Время обладает **плотностью**, которая может изменяться.
- Потоки времени формируют **динамическую структуру пространства**.
- Гравитация возникает как следствие **градиентов плотности времени**.
- Космологические процессы описываются через **временные поля**, а не через геометрию пространства.
- Время — не фон, а **активный участник физических процессов**.
